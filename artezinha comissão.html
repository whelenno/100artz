    COMISSÕES

    Descrição do Projeto

    A comissão é um trabalho personalizado encomendado por um cliente. O projeto pode ser de qualquer tipo, desde uma ilustração simples até uma obra de arte complexa.

    Nesse projeto, eu e o cliente trabalham juntos para discutir os detalhes do trabalho, incluindo o tema, o estilo e o orçamento.

    Projeto será baseado em suas especificações e requisitos, para garantir que o projeto seja exatamente o que você deseja

    Poder ser tanto para criar uma capa de música, uma ilustração de um anime/série que goste muito ou que você deseja, basta me contar o que sonha.
